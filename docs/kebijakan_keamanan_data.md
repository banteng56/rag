# Kebijakan Keamanan Data

## Tujuan
Kebijakan ini dibuat untuk melindungi data perusahaan, pelanggan, dan mitra bisnis dari kebocoran, penyalahgunaan, dan kehilangan.

## Akses Data
Akses terhadap data hanya diberikan kepada karyawan yang memiliki otorisasi resmi.
Setiap akses dicatat untuk audit keamanan.

## Perlindungan Data Pribadi
Karyawan wajib menjaga kerahasiaan data pribadi pelanggan dan dilarang membagikan informasi tanpa izin tertulis dari manajemen.

## Penggunaan Sistem
Dilarang menginstal perangkat lunak tidak resmi atau menggunakan perangkat pribadi untuk mengakses sistem internal tanpa izin dari bagian IT Security.
